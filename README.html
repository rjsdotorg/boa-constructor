Boa Constructor - An Overview

Boa Constructor is an IDE originally written in 1999, built with Python and wxPython. This project modernizes the codebase so Boa can run on current Python and wxPython releases.

Boa combines editor/debugger capabilities with a visual GUI designer for frames, menus, toolbars, dialogs, and controls. The intended development flow is Model-View-Controller, where projects are typically created and evolved directly inside Boa.

Scope: this is an active modernization effort, so some legacy features are still being brought forward to Python 3 and current wxPython Phoenix.

Installation

Basic setup steps:

  1. Install Python 3.9.1 or newer.
  2. Install wxPython 4.2.2 or newer.
  3. Download this repository and run Boa from the project root.

Installation - Windows 11

If Python is not installed, download it from python.org and enable Add Python to Path during installation.

Then install wxPython:

pip install -U wxPython

Download Boa Constructor from GitHub, extract the zip, and run Boa.py from the repository root.

You may be prompted to upgrade pip first; follow the command provided by pip.
